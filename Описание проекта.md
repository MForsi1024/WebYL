*Переводчик сленга*

Проект представляет собой навык для Яндекс Алисы, который по запросу пользователя может перевести сленговое выражение на понятный человеку язык. Через API должен подключаться Яндекс GPT. В невыке будет реализован раздел помощи. Навык должен иметь БД, с которой осуществляется работа через ORM-модель, навык хранит и обрабатывает контекст пользователя. Навык может генерировать изображение с написанием сленгового слова. Размещение навыка планируется в Яндекс Функции.
